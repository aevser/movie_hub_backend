# Movie Hub

**Movie Hub** — демонстрационный проект, созданный с целью показа навыков разработки на **Laravel**.

Проект представляет собой каталог фильмов с фильтрацией, жанрами, актёрами и режиссёрами, а также пользовательским взаимодействием (избранное, отзывы).

---

## Возможности проекта

* Каталог фильмов с пагинацией
* Просмотр фильмов по жанрам
* Детальная страница фильма
* Фильтрация фильмов по:

    * жанрам
    * актёрам
    * режиссёрам
    * рейтингу
    * дате выхода
* Поиск по названию фильма 
* Сортировка по дате и рейтингу
* Добавление фильмов в избранное
* Система отзывов
* Регистрация и авторизация пользователей

---

## Источник данных

Данные о фильмах, актёрах и режиссёрах **автоматически загружаются с внешнего API**:

**The Movie Database (TMDB)**
[https://www.themoviedb.org/](https://www.themoviedb.org/)

Используется API TMDB для получения:

* списка фильмов
* жанров
* актёрского состава
* режиссёров
* постеров и изображений

Полученные данные сохраняются в локальную базу данных и используются внутри приложения.

## Установка и запуск проекта

### Клонирование репозитория

```bash
git clone https://github.com/your-username/movie-hub.git
cd movie-hub
```

---

### Установка зависимостей

```bash
composer install
```

---

### Создание файла окружения

```bash
cp .env.example .env
```

Настройте подключение к базе данных и укажите API-ключ TMDB:

```env
MOVIE_DB_URL=https://api.themoviedb.org
MOVIE_DB_API_KEY=eafa6b6a453ad37c0002d976dff5dc97
```

---

### Генерация ключа приложения

```bash
php artisan key:generate
```

### Создание базы

```bash
touch database/database.sqlite
```

---

### Запуск миграций и тестового юзера

```bash
php artisan migrate 
php artisan db:seed

Данные пользователя:
1@1.ru  
password
```

---

### Загрузка данных о фильмах

```bash
php artisan app:import-movie-genres
php artisan app:import-movies
php artisan app:import-movie-actors
php artisan app:import-movie-crews
php artisan app:import-movie-images
php artisan app:import-movie-backdrop-images
```

### API

Проект также предоставляет собственный REST API, реализованный на Laravel и предназначенный для получения данных о фильмах.
API использует версионирование и доступен по префиксу:
```
api/v1
```

### Endpoints

```
GET /api/v1/genres - получить список жанров
```

```
GET /api/v1/movies - получить список фильмов
```

```
GET /api/v1/movies/{movie} - получить конкретный фильм
```

```
GET /api/v1/actors - получить список актеров
```
К каждому API-запросу (опционально) добавляется параметр (perPage / page) 

### Запуск локального сервера

```bash
php artisan serve
```

Приложение будет доступно по адресу:

```
http://127.0.0.1:8000/catalog
```
